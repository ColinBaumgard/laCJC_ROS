## Installation des librairies pour "TNI"

<p align="center">
xtensor : 
 
  * Download zip from : https://github.com/xtensor-stack/xtensor
 
  * Extract files and open a trminal in the folder : 
  ```bash
    $ mkdir build 
    $ cd build
    $ cmake -D CMAKE_INSTALL_PREFIX=/usr/local ..
    $ make install
    $ cd ../..
    $ rm -r xtensor-master
  ```
