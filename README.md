# laCJC_ROS
## Installation des librairies pour "TNI"
###### xtensor : 
 
  * Download zip from : https://github.com/xtensor-stack/xtensor
 
  * Extract files and open a terminal in the extracted folder : 
  ```bash
    $ mkdir build 
    $ cd build
    $ cmake -D CMAKE_INSTALL_PREFIX=/usr/local ..
    $ sudo make install
    $ cd ../..
    $ rm -r xtensor-master
  ```
